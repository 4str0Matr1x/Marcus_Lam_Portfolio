Shoot Ninja

This is a project which has taken inspiration from Fruit Ninja in p5js with a group, using a Makey Makey. I was in charge of working the Makey Makey of the game, using a makeshift gun made from cardboard. The cardboard gun is gyroscope and using it to aim and shoot with your voice. But one of my teammates made sure that it works with WASD. It's recommended to play on a quiet room as the mic sensitivity is high. Click the button below to play the game.

Fallen Grace: Redemption's Journey

A 2D, action, platformer that I’ve created in Gamemaker studio 2. I have added a few introductory levels to it. This was the first time that I have created a game in Gamemaker Studio, by myself before. Using the Gamemaker Studio 2's tutorials, it has taught me a lot about the fundamentals of game design and how the coding works. I decided to work on a platformer game first because I wanted to start off with something that has a lot of tutorials and help me understand how Gamemaker Studio works.
